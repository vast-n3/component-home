Welcome to vast-n3

You can jump right into development! vast-n3 is as non-invasive as it gets and can be deployed "as is".
May we suggest to familiarize yourself with the following:

Tailwind neoan3 axios

It's basically tailwind

The utility-first css framework tailwind gives you total control over your styles. vast-n3 uses postCSS and relevant files can be found in "frame/vastn3/css".

Theming

vast-n3 adds the prefixes "primary", "secondary", "success", "warning" and "danger" to tailwind. You can change these values in "frame/vastn3/css/theme.css"

Building

Since vast-n3 uses postCSS as a preprocessor, please refrain from using the tailwind commands. Instead, run npm build

neoan3

The PHP multi purpose framework neoan3 provides a flexible basis for your API & CRUD needs.

Setup

We have taken care of most of it for you, but you will probably verify your database credentials now:
neoan3 credentials
Please make changes to "vastn3_db" accordingly. Then, run
neoan3 migrate models up

NOTE: The first time you run this command, the tool will ask you for your credentials again. As you can host & run multiple credentials, the cli tool saves its own credentials for operations.

Your first component

We created templates in "_template" optimized for vast-n3. Let's build our first component.

neoan3 new component myComponent

Choose "Custom Element". The cli tool has not created files in the folder "component/myComponent". In order to make this component available on a route, add it to the $vueComponents-array.

NOTE: Routing endpoints are defined in "component/[name]/[name].ctrl.php"'s init-function. You can change the default route (default_ctrl) in the file "default.php".

You can now open the used template file (usually "component/[name]/[name].view.html") and use your component:
<my-component></my-component>

axios & the api

We have wrapped axios in a globally available variable "api". No don't need to import or define anything else. In your component, the code api.get('home') will target the function "getHome" located in "component/home/Home.ctrl.php"

Example

component/myComponent/myComponent.ce.js

mounted(){
    api.get('home?test=works').then(res => {
        console.table(res.data)
    })
}

component/home/Home.ctrl.php

function getHome($parameters = [])
{
    return [
        'msg' => 'API response',
        'sent_params' => $parameters
    ];
}
